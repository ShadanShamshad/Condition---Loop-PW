Loops are used in programming to repeatedly run a block of code.

For and while loop are use interchangeably, although they have different syntax they have equal capability, developer use different loops in different scenarios.

We use for loop to run a piece of code until the set condition turns false.

There are different types of loops in JavaScript.

  1. For Loop:

    SYNTAX

    for (initial expression; condition; update expression) {
    // for loop body
    }

    Example

    for (let i = 0; i < 3 ; i++) {
    let name = "PW Skills";
    console.log(name);
    }

    // Output
    / *
    PW Skills
    PW Skills
    PW Skills
    * /

    In the above loop we run the loop until the value of i becomes false, and print PW Skills as long as the loop runs(i is true)

  2. While Loop:

    SYNTAX

    while ( condition ) {
    // body of the loop
    }

    Example

    let i = 1, n = 4;
    while (i <= n ) {
    console.log(i);
    i = i + 1;
    }

    // Output
    / *
    1
    2
    3
    4
    * /

    This loop keeps on running as long as the condition is true.

  3. Do While Loop:

    SYNTAX

    do {
    // body of the loop
    } while (condition);

    Example

    let result = ' ';
    let i = 0;
    do {
    i = i + 1;
    result = result + i;
    } while (i < 7);

    console.log(result);
    // Output
    / *
    1234567
    * /

    Here we write a program to print numbers from 1 to 7 in a line using do-while loop
